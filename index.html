El Impostor

Cómo se juega

El juego de las palabras y la desconfianza

Modo clásico: todos reciben la misma palabra secreta, excepto el impostor, que recibe una pista.

Modo fútbol: todos reciben el nombre de un futbolista famoso, excepto el impostor, que solo sabe que es el impostor.

Cada uno dice algo relacionado con su palabra. El grupo debate y vota. Si fallan, ese jugador queda eliminado y se vota de nuevo. El impostor gana si llega a quedar solo con otro jugador.
